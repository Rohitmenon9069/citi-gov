What is CitiGov?

Rebuilding trust between 1.4 Billion citizens and their government

CitiGov is India's first open Digital Trust Analytics Dashboard — a real-time platform that measures, visualises, and helps improve citizen confidence in India's digital government services including Aadhaar, DigiLocker, UMANG, UPI, CoWIN, and 1,700+ other e-governance platforms.

Build Journey
Step 1
Research & Data Curation
Sourced real 2025 data from Digital India Foundation, NASSCOM, UIDAI, MeitY and NASSCOM Digital Trust surveys across 42,000+ respondents in 28 states.
Step 2
Design System
Built a complete CSS design token system — navy palette, glassmorphism cards, glow effects, micro-animations — inspired by modern gov-tech dashboards from Singapore and Estonia.
Step 3
Interactive Components
Built 7 feature sections: Trust Scorecard, Pillar Cards, Reality Gap Journey Map, Barrier Quadrant, Adoption Simulator, Case Studies, and a Drag-and-Drop Action Planner.
Step 4
Auth System & Accounts
Added a full Login + Create Account flow with localStorage-based user storage, password strength meter, field validation, and session persistence — no backend needed.
Step 5
Accessibility & Polish
WCAG 2.1 AA compliance: semantic HTML, keyboard navigation, ARIA live regions, focus rings, responsive design for mobile, tablet, and desktop viewports.
